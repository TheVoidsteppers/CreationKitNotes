
# 安装方式

```shell
# 1、解压 CreationKit1.5.73.rar 或 CreationKit1.5.73.zip 里的文件到游戏根目录
# 2、解压 CK64Fixes Release 3.2-20061-3-2-1616560054.zip 和 Creation Kit Fix Version 2.0-47635-2-0.rar 到游戏根目录（这两个是修复 ck bug 和 提高运行速度的）
# 3、若要支持中文输入，去解压 Creation Kit 64 SSE Unicode 下的其中一个压缩包到游戏根目录

# 到此 ck 安装完成，若要测试 mod 还需使用 Mod Organizer 管理 mod
```
